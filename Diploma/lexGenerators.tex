\subsection{Генератор лексических анализаторов Lex} \label{sub116}

\textit{Lex} (в~более поздних реализациях \textit{Flex})~--- программный инструмент, который позволяет определить лексический анализатор, указывая регулярные выражения для описания шаблонов токенов. Входные обозначения для Lex обычно называют \textit{языком Lex}, а~сам инструмент~--- \textit{компилятором~Lex}. Компилятор~Lex преобразует входные шаблоны в~диаграмму переходов.
